## 2. Description of the Service

Stellar Cargo is a non-custodial, browser-based Progressive Web App that allows users to:

- Create and manage multiple Stellar network accounts
- Send and receive Stellar Lumens (XLM) and other Stellar-based assets
- Perform chunked and batch transactions
- Split and merge token balances across accounts
- Back up and restore wallet data using Google Drive
- Import and export account data

Stellar Cargo operates as a front-end interface that communicates directly with the publicly accessible Stellar network. We do not operate any proprietary blockchain, custody any funds, or provide financial services.
